# 长文图解/知识卡片生成器

## 概述

将长文章或主题自动转化为精美的手绘风格信息图，适合社交媒体分享。全自动四步流程：信息提取 → 主题规划 → 视觉设计 → 图像生成。

## 工作流程决策树

```
用户请求
    │
    ├─ 直接提供文章 → 执行完整四步流程
    ├─ 提供主题关键词 → 先生成文章内容，再执行四步流程
    ├─ 提供名言/金句 → 生成单张名言卡片
    └─ 提供学习笔记 → 提炼知识点，生成卡片
```

## 核心设计原则

**视觉风格**：手绘风格，温暖亲和，高信息密度

- 背景：米白色 #F9F7F2（带纸张纹理）
- 主文字：暖炭灰 #3E3C38
- 强调色：珊瑚红 #FF7F7F、鼠尾草绿 #8FA87A、温暖黄 #FFD166、天空蓝 #6FBADC

**信息原则**：多用图形，少用纯文字；每张图独立可理解；整体连贯形成叙事

**署名要求**：所有卡片必须包含「BY冻柠AI」署名

## Step 1: 深度阅读与信息提取

### 任务
通读2000-3000字文章，提取核心论点、关键论据、重要数据和典型案例。

### 执行逻辑

1. **全文通读**：理解核心观点和写作意图，识别文章结构
2. **核心信息提取**：
   - 中心论点（1-2句话）
   - 主要论据（3-6个）
   - 重要数据（数字、百分比、对比）
   - 典型案例（例子、故事）
   - 关键概念（重点术语）

3. **逻辑关系梳理**：分析各论据关系（并列、递进、因果）

### 输出格式

```
【文章核心论点】
[1-2句话概括]

【主要论据】
1. [论据1]
2. [论据2]
...

【重要数据】
- [数据1]
- [数据2]
...

【典型案例】
- [案例1]
- [案例2]
...

【视觉化建议】
[哪些内容适合用图形、图标、图表呈现]
```

---

## Step 2: 主题分组与张数规划

### 任务
将内容按逻辑分组为3-6个独立主题，每个主题对应一张信息图。

### 确定图片张数

- **3张图**：结构简单、论点集中的文章
- **4-5张图**：中等复杂度、多个论点（最常见）
- **6张图**：信息量大、多维度分析

**判断标准**：每张图承载一个独立主题，3-5个支撑要点，信息密度适中

### 主题分组策略

**A. 递进式**（逻辑链条）
- 图1：问题/现状 → 图2：原因/分析 → 图3：解决方案 → 图4：实施步骤 → 图5：预期效果

**B. 并列式**（多维度）
- 图1：总览框架 → 图2-5：各维度展开 → 图6：总结

**C. 金字塔式**（观点类）
- 图1：核心观点 → 图2-4：支撑论据 → 图5：案例数据 → 图6：结论建议

### 输出格式

```
【总体规划】
- 图片总数：[3-6]张
- 分组逻辑：[递进式/并列式/金字塔式]
- 叙事主线：[一句话描述整体逻辑]

---

【图1】
标题：[5-10字]
核心观点：[15-25字]
支撑要点：
  1. [要点1，10-20字]
  2. [要点2，10-20字]
  3. [要点3，10-20字]
视觉元素建议：[建议的图形、图标类型]

【图2】
[继续其他图片...]

---

【整体连贯性说明】
[说明各图之间的逻辑关系]
```

## Step 3: 视觉元素设计

### 任务
为每张图设计具体的视觉元素（图形、图标、简笔画、图表）。

### 布局方式选择

**A. 中心放射型**（核心观点 + 多个支撑要点）
- 中心：大标题 + 核心观点（配主视觉图形）
- 周围：支撑要点环绕

**B. 上下分层型**（明确层级）
- 上层：标题 + 核心观点
- 中层：支撑要点分区
- 下层：署名「BY冻柠AI」

**C. 左右分栏型**（对比或并列）
- 左侧：标题 + 核心观点
- 右侧：视觉化图表

**D. 流程型**（时间或逻辑顺序）
- 用箭头连接各环节

### 视觉元素库

**数据类**：手绘柱状图、饼图、折线图、上升/下降箭头、百分比圆环

**概念类**：
- 灯泡（创意）、火箭（增长）、齿轮（系统）
- 天平（平衡）、靶心（目标）、钥匙（关键）

**流程类**：箭头、时间线、步骤圆圈

**人物类**：简笔画小人（工作、思考、协作）、表情图标

### 输出格式

```
【图1视觉设计方案】

布局方式：[中心放射型/上下分层型...]

具体元素设计：
1. 标题区域：文字、视觉处理、位置
2. 核心观点区域：文字、主视觉图形描述、位置
3. 支撑要点区域：每个要点的配套图标、颜色、位置
4. 署名区域：「BY冻柠AI」，位置

配色方案：背景、主文字、强调色
视觉风格关键词：[3-5个关键词]
```

## Step 4: 生成信息图

### 任务
使用imagen skill生成所有信息图（3-6张）。

### Prompt编写结构

每张图的prompt必须包含：

```
[整体风格描述]
手绘风格信息图，温暖亲和，米白色背景（#F9F7F2），自然手绘线条，配色柔和（暖炭灰 #3E3C38 + 珊瑚红 #FF7F7F + 鼠尾草绿 #8FA87A + 温暖黄 #FFD166 + 天空蓝 #6FBADC）。

[布局描述]
[上下分层型/中心放射型/左右分栏型/流程型]布局，[具体空间分配]

[具体内容元素]
1. 顶部：手写风格标题「[标题]」，[装饰元素]
2. 中心/上部：核心观点「[观点]」，[主视觉图形详细描述]
3. 支撑要点区域：
   - 左侧：图标[描述] + 文字「[要点]」，使用[颜色]
   - 中间：图标[描述] + 文字「[要点]」，使用[颜色]
   - 右侧：图标[描述] + 文字「[要点]」，使用[颜色]
4. 右下角：署名「BY冻柠AI」，[装饰元素]

[风格强调]
整体风格：手绘笔记本风格，线条自然有波浪感，简笔画风格，温暖亲和，信息密度高，多用图形少用纯文字。
```

```
- 质量：high
```

```
📊 图片概览：
```

## 特殊场景处理

### 场景A：主题生成
用户输入主题（如"量子物理入门"）→ AI生成2000字科普文章 → 执行Step 1-4

### 场景B：名言金句
生成单张卡片：
- 布局：上下分层型
- 中心：名言大字（手写字体）
- 背景：简洁装饰图案
- 底部：出处 + 「BY冻柠AI」

### 场景C：学习笔记
侧重公式、概念解释：
- 多用图表展示概念关系
- 公式用手绘风格呈现
- 关键术语用图标标注

### 场景D：文章提炼
用户提供文章URL或文本 → 直接执行Step 1-4

## 质量检查清单

执行每个步骤后，检查：

**Step 1**：
- [ ] 核心论点是否准确？
- [ ] 主要论据是否完整？
- [ ] 是否遗漏关键信息？

**Step 2**：
- [ ] 图片总数是否合理（3-6张）？
- [ ] 每张图能否独立理解？
- [ ] 整体是否形成完整叙事？

**Step 3**：
- [ ] 每个要点是否配有视觉元素？
- [ ] 风格是否统一（手绘、温暖）？
- [ ] 是否包含「BY冻柠AI」署名？

**Step 4**：
- [ ] 是否成功生成所有图片？
- [ ] 手绘风格是否到位？
- [ ] 信息密度是否足够？
- [ ] 文字是否清晰可读？

**最佳输入**：2000-3000字的结构清晰、论点明确的文章

**适用内容**：知识科普、行业分析、观点评论

**不适用内容**：纯叙事性散文、缺乏结构化信息的文章

- 图片卡片（PNG/JPG，社交媒体分享）
- HTML网页（交互式在线展示）
