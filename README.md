
## 技能系统

Claude Skills 是一种扩展 Claude Code 能力的方式，每个技能都是一个独立的目录，包含：
- **SKILL.md** - 技能定义文件，包含名称、描述、元数据
- **scripts/** - 实现技能功能的脚本
- **assets/** - 相关资源文件
- **references/** - 参考文档
