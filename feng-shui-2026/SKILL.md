# Feng Shui 2026 - 丙午年九宫飞星风水布局

## 概述

此技能提供2026年（丙午火马年）九宫飞星风水布局的完整解决方案，支持户型分析和个性化风水布局建议生成。

## 核心功能

### 1. 户型分析
- 支持图片格式户型图（PNG/JPG）
- 支持文字描述户型
- 识别房间位置、大门朝向、方位系统

### 2. 九宫飞星布局
- 2026年丙午年飞星分布
- 各方位吉凶判断
- 飞星与房间对应关系

### 3. 个性化布局建议
- **催婚/桃花**：粉水晶、鲜花、成双成对摆件
- **催事业/财运**：发财树、聚宝盆、富贵竹
- **催健康**：绿色植物、保持��风明亮
- **催学业/文昌**：文昌塔、毛笔四支
- **化解凶煞**：铜铃铛、五帝钱、铜葫芦

### 4. 导出格式
- CSV格式表格（可导入飞书）
- Markdown格式文档
- 分类清晰的布局清单

## 工作流程

### 步骤1：获取户型信息
```
询问用户提供：
- 户型图（图片或文字描述）
- 方位系统（如：上北下南）
- 各房间位置
- 居住者及需求
```

### 步骤2：分析飞星布局
```
根据用户提供的方位系统：
1. 确定各房间对应的飞星
2. 标记吉凶方位
3. 识别需要化解的凶位
```

### 步骤3：生成布局方案
```
按房间生成布局建议：
- 吉位：催旺物品和方法
- 凶位：化解物品和方法
- 个性化：根据用户需求定制
```

### 步骤4：生成CSV表格
```
生成包含以下字段的CSV：
住宅方案,房间,方位,飞星,吉凶,居住者,需求,类型,物品,作用,摆放位置
```
